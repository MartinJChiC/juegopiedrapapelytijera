Piedra, papel, tijera

Selecciona tu jugada:

Este código HTML contiene tres botones que representan las jugadas posibles (piedra, papel y tijera), y un campo de texto para mostrar el resultado. Cuando el usuario hace clic en uno de los botones, se llama a la función jugar con el valor de la jugada seleccionada. La función jugar genera una jugada aleatoria para la computadora y compara la jugada del usuario con la de la computadora para determinar el resultado. El resultado se muestra en el campo de texto. Espero que esto te sea útil. ¡Diviértete jugando!
